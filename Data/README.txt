
To export data from database tables

	Run : ExportBaseline {folderPath} {ServerName} {DataBaseName}

	Example : ExportBaseline ..\DummyData .\SQLExpress "[HomeKookd.main]"
	
To import data into database tables

	Run : ResetBaseline {folderPath} {ServerName} {DataBaseName}

	Example : ResetBaseline ..\DummyData .\SQLExpress "[HomeKookd.main]"
